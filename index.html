CategoriesProgramming, Rust
Tagsrust C examples mutation variable meta programming
For a first programming post, I have chosen a beginner-friendly topic, regarding Rust’s mut keyword: What does mut truly mean? what does it do? For starters, let’s quote TRPL1: Mutability, the ability to change something Hum, it looks like they got it the other way around. Let’s fix it: Mutability, the ability to change something Mutability, the property of being change-able Note the change in the grammatical aspect: if a mutator modifies a mutatee, the mutability required for it to be possible is the mutatee’s one (and not the mutator’s).
